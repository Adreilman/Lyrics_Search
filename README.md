# 🎶 Lyrics Search App

A simple lyrics search application built with React, Node.js, and Express.

## 🚀 Features
- Search for any song's lyrics
- Display lyrics fetched from an external API
- Responsive UI

## 🛠️ Tech Stack
- React
- Node.js
- Express

## 📦 How to Run
```bash
git clone https://github.com/Adreilman/Lyrics_Search.git
cd Lyrics_Search
npm install
npm start
